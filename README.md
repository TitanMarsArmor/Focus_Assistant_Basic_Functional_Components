# 智能专注助手 - 任务2基础功能部件

本目录包含了任务2的基础功能部件实现，基于多模态情境感知的智能专注助手项目。

## 文件结构

- `volume_monitor.py` - 音量监控助手，检测到音量不为0且有媒体播放时自动暂停并提示
- `hotkey_controller.py` - 快捷键控制器，使用A键暂停视频，B键切换窗口
- `document_saver.py` - 文档自动保存助手，使用A键快速保存当前活动文档
- `smart_assistant.py` - 智能专注助手主程序，整合以上所有功能

## 功能说明

### 1. 音量监控助手
- 自动检测系统音量状态
- 当检测到音量不为0且有媒体播放时，自动暂停媒体播放
- 显示弹窗提示用户当前为外放状态
- 按Q键退出程序

### 2. 快捷键控制器
- 全局快捷键控制，无论焦点在哪个窗口都能响应
- 按A键：暂停/播放当前视频
- 按B键：切换到上一个窗口（Alt+Tab功能）
- 按Q键：退出程序

### 3. 文档自动保存助手
- 智能识别当前活动窗口的应用程序类型
- 按A键：自动保存当前活动文档
- 支持大多数常见的办公软件和编辑器
- 实时显示保存状态和结果
- 按Q键退出程序

### 4. 智能专注助手主程序
- 整合以上所有功能组件
- 可单独或批量启动各个功能
- 提供友好的用户界面
- 实时状态显示
- 一键停止所有功能

## 安装依赖

运行本程序需要安装以下Python库：

```bash
pip install pynput pyautogui psutil pygame pywin32
```

## 使用方法

### 方法一：使用整合的主程序（推荐）

1. 确保已安装所有必要的依赖
2. 运行主程序：
   ```bash
   python smart_assistant.py
   ```
3. 在弹出的界面中选择要启动的功能
4. 点击"启动选中功能"或"启动全部功能"

### 方法二：单独运行各个功能组件

可以单独运行每个功能组件：

```bash
# 运行音量监控助手
python volume_monitor.py

# 运行快捷键控制器
python hotkey_controller.py

# 运行文档自动保存助手
python document_saver.py
```

## 注意事项

1. 程序需要一定的系统权限才能控制音量、模拟键盘输入等操作

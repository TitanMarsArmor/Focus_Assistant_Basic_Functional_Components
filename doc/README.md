# 智能专注助手 - 文档目录

本目录包含智能专注助手项目的相关文档，旨在帮助开发者和用户理解项目的设计、架构和功能。

## 文档列表

### 1. 项目说明文档.md

详细的项目说明文档，包含以下内容：
- 项目概述和核心交互理念
- 设计需求分析
- 系统架构设计
- 核心功能模块详解
- 核心流程说明
- 技术栈和依赖
- 使用说明
- 未来扩展建议

### 2. class_diagram.puml

UML类图，展示系统的类结构和它们之间的关系。可以使用PlantUML工具查看或转换为图片。

### 3. flow_diagram.puml

UML流程图，展示系统的主要工作流程，包括：
- 系统启动流程
- 音量监控流程
- 快捷键操作流程

## 查看UML图

要查看.puml文件中的UML图，您可以：

1. 使用支持PlantUML的IDE插件，如VSCode的PlantUML插件
2. 在线PlantUML渲染器，如https://www.planttext.com/
3. 安装PlantUML本地工具并生成图片

## 文档更新说明

这些文档随着项目的发展可能需要更新。请在修改代码后同步更新相关文档，以保持文档与代码的一致性。
